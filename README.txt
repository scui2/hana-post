# HANA Post

HANA Post is an advanced Foundation 6 theme that is perfect for your news and magazine sites. It is fully integrated with <a href="http://rewindcreation/hana-block/">Hana Block</a> that allows you to design flexible and professional news site without coding.

## Documentation

For documentations and user guides, visit http://rewindcreation.com/docs

* Jetpack is activated
* and there is at least ONE post tagged "featured" or user defined tag.
